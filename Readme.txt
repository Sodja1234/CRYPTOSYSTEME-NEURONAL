Pour éxécuter et voir l'execution du cryptosystème neuronal :
- Assurez-vous d'avoir installer les bibliothèques Python ci-dessous :
    * Pillow
    * Math
    * Time
    * Random
    * PyQt
    * Numpy
- Ouvrer le fichier PerceCrypt pour observer le chiffrement et le dechiffrement 

Pour réaliser les analyses par histogramme et par variances :
- Assurez-vous d'avoir installer les bibliothèques Python Matplotlib
- Exécuter le fichier AnalyseRéaliser
